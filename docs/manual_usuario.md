# Manual de usuario

## Iniciar sesion

Ingresar con el usuario y contrasena configurados en el sistema.

## Procesar facturas

Desde la seccion de procesamiento se puede cargar una factura en PDF o imagen para extraer datos y registrarla.

## Revisar facturas

La seccion de facturas permite consultar, validar y actualizar la informacion registrada.

## Reportes

La seccion de reportes muestra resumenes y datos utiles para el control de facturas.

## Backup

El sistema permite exportar una copia de seguridad desde configuracion.
